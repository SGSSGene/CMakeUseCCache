# CMakeUseCCache

Easy use of ccache in combination with cmake and CPM.

It will provide a new cmake option `USE_CCACHE` which is by default `ON`, but can be turned of by users.

```
CPMAddPAckage("gh:SGSSGene/CMakeUseCCache@1.0.0")
```
